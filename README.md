# Dashboard de Vendas do Xbox

## 🎯 Objetivo

Este projeto tem como objetivo criar um dashboard interativo e visualmente atrativo no Excel para análise de vendas do Xbox. O dashboard transforma dados brutos em informações claras, facilitando a tomada de decisões estratégicas com base em dados reais.

---

## 🗂️ Dados Utilizados

- **base.xlsx**: Arquivo com os dados brutos de vendas do Xbox, contendo informações como data da venda, produto, quantidade, valor, região, etc.
- **dashboard_xbox_finalizado.xlsx**: Arquivo Excel com o dashboard construído, pronto para análise.

---

## 📊 Como Reproduzir o Dashboard

1. **Faça o download dos arquivos `base.xlsx` e `dashboard_xbox_finalizado.xlsx`.**
2. Abra o arquivo `dashboard_xbox_finalizado.xlsx` no Microsoft Excel (ou compatível).
3. Navegue pelas abas para visualizar:
   - Resumo geral de vendas
   - Gráficos de desempenho por período, produto e região
   - Indicadores-chave (KPIs)
4. Caso queira construir seu próprio dashboard:
   - Importe os dados de `base.xlsx` em uma nova planilha do Excel.
   - Utilize Tabelas Dinâmicas e Gráficos para criar visualizações.
   - Personalize o layout conforme sua preferência.

---

## 💡 Dicas para Análise

- Observe tendências de vendas ao longo do tempo.
- Compare o desempenho entre diferentes regiões e produtos.
- Analise os principais produtos vendidos e oportunidades de crescimento.
